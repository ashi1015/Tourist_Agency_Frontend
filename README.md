# Tourist Agency Frontend - Destination Management


React Js, MongoDB, Node Js, Express Js

CRUD, Search, Pagination

### Set port
.env
```
PORT=8082
```

### Project setup

In the project directory, you can run:

```
npm install
# or
yarn install
```

or

### Compiles and hot-reloads for development

```
npm start
# or
yarn start
```

Open [http://localhost:8082](http://localhost:8081) to view it in the browser.

The page will reload if you make edits.
